# Secure OSINT CRM — Offline Encrypted Personal Knowledge Base

Your private, zero-cloud, fully encrypted desktop app for:
- Legal OSINT, investigative research, due-diligence, journalism, or small-business sales.

Features
- 100% offline — nothing ever leaves your computer  
- End-to-end encrypted SQLite (SQLCipher)  
- Full-text search across people, companies & notes  
- Obsidian-style `[[Wiki Links]]`  
- Kanban sales/investigation pipeline  
- Timeline view of a target’s digital footprint  
- Markdown editor with live preview  
- Windows · macOS · Linux (15 MB app)

Perfect for
- Licensed private investigators
- Journalists protecting sources
- Sales teams who hate SaaS lock-in
- Anyone who values privacy
